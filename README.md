# Moonlit Hearts 🌙💖

A romantic, interactive glassmorphism single-page application built as a personal gift. This project focuses heavily on emotional storytelling and advanced modern web design.

## Features ✨

- **Advanced Glassmorphism UI**: Layered glass cards, floating glass panels, aurora gradients, and 3D hover tilt effects.
- **Story-Based Journey**: A smooth flow from the Welcome Portal to a Proposal Ending.
- **Relationship Dashboard**: Live updating counter tracking the exact days, hours, minutes, and seconds since April 8th, 2019.
- **Memory Gallery**: Beautiful 3D polaroid gallery featuring personal memories.
- **Little Love Games**: Interactive dates generator and an adventure bucket list.
- **Hidden Secret Notes**: Scattered clickable stars revealing 20 different secret sweet messages.
- **Custom Ambient Audio**: Toggle between Piano, Night Ambience, or Romantic Instrumental.

## Technologies Used 💻

- **React 18**
- **TypeScript**
- **Vite**
- **Tailwind CSS v4**
- **Framer Motion** (for smooth animations and transitions)
- **Lucide React** (for icons)

## Getting Started 🚀

1. Clone the repository:
   ```bash
   git clone https://github.com/devlinuskibet/cee.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open your browser and visit `http://localhost:5173`
